# GBCC
#### A simple C compiler for the Gameboy's GBz80 processor

## Installation
- Install Ocaml
- Build executable
```
make
```
- Install executable to local bin
```
make install
```
- To uninstall
```
make uninstall
```
- To clean src directory
```
make clean
```

## Usage
```
gbcc path/to/dest.asm path/to/src.c
```
